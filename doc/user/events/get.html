Events: List

Description

List all the Events available in this Environment.

HTTP Request

GET /api/v1beta0/user/envId/events/

Parameters

Name Required Description Location
envId
integer		 Yes

The ID of the Environment scoping this request.

 path

Request body

Do not supply a body with this request.

Successful Response (Status 200)

A list of Event objects.

The API replies with a EventListResponse object. Example response body:
